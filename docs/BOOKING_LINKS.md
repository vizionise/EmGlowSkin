# EmGlowSkin Booking Link

Use this official booking URL for all site booking calls to action.

| Purpose | URL |
| --- | --- |
| Square booking page | `https://emglowskin.square.site/` |

## Usage Notes

- Use this link for buttons labeled `Book`, `Book Now`, and `Book Your Glow`.
- Keep the main navigation `Contact` link pointed to `contact.html`.
- Use `target="_blank"` and `rel="noopener"` for the external booking link.
